# {Coders handbook} Text Editor - Tauri 2.0

![App Screen Shot](src-tauri/app-icon.png)

A high-performance, professional-grade text editor built with the **Tauri 2.0** framework. This editor features a unique **Terminal/Matrix-inspired aesthetic**, integrated with powerful modern web technologies for a lightning-fast experience.

## ‚ú® Features

- **Terminal Aesthetic**: High-contrast Matrix green-on-black theme with CRT scanline effects.
- **Tauri 2.0 Kernel**: Native performance using Rust backend and modern Vite-powered frontend.
- **Monaco Editor Core**: The same powerful editor engine that fuels VS Code, including full syntax highlighting for dozens of languages.
- **Native File System**: Direct interaction with your local file system via Tauri's secure `fs` plugin.
- **Workspace Explorer**: Mount your local directories and manage project files seamlessly.
- **Customizable Experience**: Adjustable font sizes, tab settings, word wrap, and auto-save persistence.
- **Responsive Layout**: Fluid UI with configurable sidebars and multi-tab support.

## üöÄ Tech Stack

- **Core Framework**: [Tauri 2.0](https://tauri.app/)
- **Frontend**: [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)

- **Icons**: [Lucide React](https://lucide.dev/)

## üõ†Ô∏è Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (latest LTS)
- [Rust](https://www.rust-lang.org/tools/install)
- [Tauri Build Toolchain](https://tauri.app/v1/guides/getting-started/prerequisites)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/[your-username]/text-editor-v2.git
   cd text-editor-v2
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run in development mode:
   ```bash
   npm run tauri dev
   ```
